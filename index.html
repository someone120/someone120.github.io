quarkmc
2020年03月19日 119 字 大概 1 分钟

Quarkmc 是一个用于 Minecraft Bedrock Edit 的 ws 服务器。他的存在让 Minecraft Bedrock Edit 大大增加了扩展性。

安装

1.克隆或者下载 zip 包。

2.移动到目录

3.执行 npm install

插件

插件是 Quarkmc 的灵魂。如果没有了插件，就像人类失去了智力。

安装插件

1.下载插件并放在 Quarkmc 的目录中。

2.执行 node plugin.js install 插件的名称。
